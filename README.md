## Architecture

Backend:
- Laravel - RESTful API for business logic
- MySQL – Relational database for storing users, trips, and expense data
  
Frontend:
- React – Interactive, responsive user interface
- Tailwind CSS – Modern, customizable styling framework

## Description
A monolithic application with frontend and backend for managing contacts. Some of the functionalities of the project:
- Database of companies and emloyees - basic information, maps, contact history,
